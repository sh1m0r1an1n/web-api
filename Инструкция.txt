1. Создание виртуального окружения

cd C:\Users\ВашеИмя\MyProject # Перейти в папку проекта

python -m venv venv # Создать виртуальное окружение

2. Активация виртуального окружения

venv\Scripts\activate

3. Установка библиотек в виртуальное окружение

pip install requests # Пример для requests

pip list # Список установленных пакетов

4. Деактивация виртуального окружения

deactivate # Командная строка вернется в обычный режим, и Python будет использовать глобальные библиотеки
